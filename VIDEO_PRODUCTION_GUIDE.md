# Video Production Guide for Focal Landing Page

## Overview
This guide provides prompts and instructions for creating product demo videos for the Focal landing page. These videos showcase the keyboard-first productivity features with professional, engaging content.

## Video Specifications
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Frame Rate**: 30fps or 60fps
- **Duration**: See individual video requirements
- **Style**: Clean, modern, professional with smooth transitions
- **Audio**: Background music (subtle), keyboard sound effects, voiceover optional

## Tools Recommended
- **Screen Recording**: OBS Studio, CleanShot X, Loom
- **Video Editing**: DaVinci Resolve (free), Final Cut Pro, Adobe Premiere
- **Cursor Effects**: Mouseless (for cursor highlighting), CursorFX
- **Keyboard Visualization**: Keycastr (Mac), KeyPress OSD (Windows)

## Video 1: Hero Section - Product Overview
**Duration**: 60-90 seconds
**Purpose**: High-level product demo showing the main value proposition

### AI Prompt for Video Generation (if using AI video tools):
```
Create a 75-second screen recording demo of a productivity application called Focal. Show:
1. Clean dashboard interface with task list and calendar view
2. User pressing keyboard shortcuts (T+B) to create time blocks
3. Smooth cursor movements and keyboard typing animations
4. Tasks being completed with keyboard navigation
5. Professional, modern UI with blue accent colors
6. Smooth transitions between different views
7. End with the Focal logo and tagline "Zero-Click Productivity"
```

### Manual Recording Script:
1. **0:00-0:10**: Show Focal dashboard with task list
2. **0:10-0:25**: Press T+B to create a time block, show the modal
3. **0:25-0:40**: Navigate tasks using arrow keys, show selection
4. **0:40-0:55**: Complete a task with Space bar, show it move to completed
5. **0:55-0:75**: Show calendar view with time blocks, smooth transitions

### Key Visual Elements:
- Highlight keyboard shortcuts with on-screen key display
- Show cursor only when necessary (keyboard-first emphasis)
- Use smooth zoom effects for focus areas
- Add subtle blue glow effects on active elements

## Production Tips

### Recording Setup
1. **Clean Environment**: Close unnecessary apps, clear desktop
2. **Consistent Theme**: Use light mode for better visibility
3. **Font Size**: Increase to 14-16px for readability
4. **Window Size**: Record at 1280x720 or 1920x1080
5. **Hide Personal Data**: Use demo account with fake data

### Keyboard Visualization
- **Mac**: Keycastr (free, shows key presses on screen)
- **Windows**: KeyPress OSD (free alternative)
- **Cross-platform**: Use video editing to add key overlays

### Editing Guidelines
1. **Smooth Transitions**: Use 0.3-0.5s fade transitions
2. **Zoom Effects**: Subtle zoom on important actions (1.1x-1.2x)
3. **Speed**: Keep at 1x speed, don't speed up
4. **Cursor**: Hide cursor during keyboard-only actions
5. **Audio**: Add subtle keyboard click sounds

### Color Grading
- Match Focal's Royal Blue (#2563EB) accent
- Keep colors vibrant but professional
- Ensure good contrast for accessibility
- Use consistent color scheme across all videos

### Export Settings
- **Codec**: H.264
- **Bitrate**: 5-10 Mbps for 1080p
- **Audio**: AAC, 128kbps, 44.1kHz
- **Container**: MP4
- **Profile**: High

## File Naming Convention
- `hero-demo.mp4` - Hero section video
- `time-blocking-shortcuts.mp4` - Time blocking feature
- `ai-scheduling.mp4` - AI scheduling feature
- `task-templates.mp4` - Templates feature
- `subtasks-demo.mp4` - Subtasks feature

## Placement in Code
Update video paths in `FeatureVideo.tsx`:
```tsx
<FeatureVideo
  videoSrc="/videos/hero-demo.mp4"
  poster="/videos/hero-poster.jpg"
  title="Focal Product Demo"
/>
```
